## 기능 목록
1️⃣ 정답 숫자 생성
<br>
2️⃣ 입력 숫자에 따른 볼, 스트라이크 수 출력
<br>
3️⃣ 정답을 맞출 때까지 입력 숫자를 반복
<br>
4️⃣ 입력 숫자에 따른 재시작 & 종료 결정

## 예외 목록
- 사용자의 입력값

    <br>
  
    - 숫자 입력
  
    1️⃣ 문자를 포함해 입력했을 때
    <br>
    2️⃣ 3자리 숫자를 입력하지 않았을 때
    <br>
    3️⃣ 아무것도 입력하지 않았을 때
    <br>
    4️⃣ 중복된 숫자를 입력했을 때

    <br>

    - 재시작 숫자 입력

    1️⃣ 1, 2가 아닌 숫자를 입력했을 때
    <br>
    2️⃣ 문자를 입력했을 때
    <br>
    3️⃣ 아무것도 입력하지 않았을 때

<br>
- 정답 숫자 생성

1️⃣ 중복된 문자가 존재할 때
